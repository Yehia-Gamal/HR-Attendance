# تقرير إصلاح تسجيل الدخول والبصمة وتنسيق صفحة المجمع

تاريخ التنفيذ: 27 أبريل 2026

## ما تم إصلاحه

1. **تنسيق صفحة بصمة الموظف**
   - منع ظهور رابط Google Maps الطويل داخل الكارت.
   - إضافة زر منفصل لفتح المجمع على Google Maps.
   - ضبط كروت الصفحة حتى لا تتداخل الأعمدة ولا تكسر العرض.
   - تحسين العرض على الموبايل والتابلت.

2. **مشكلة رفض البصمة بسبب دقة GPS**
   - رفع الحد العملي لدقة GPS إلى 2000 متر للاختبار والتشغيل الواقعي.
   - إعادة محاولة قراءة الموقع تلقائيًا مرة ثانية عند ضعف الدقة.
   - إضافة حالة قبول جديدة: inside_branch_low_accuracy.
   - إذا كان الموقع داخل نطاق المجمع مع هامش الدقة لا يتم رفض البصمة تلقائيًا.

3. **تسجيل الدخول عند الخطأ**
   - عند إدخال بيانات خاطئة لا يتم استبدال البريد أو كلمة المرور بما كان افتراضيًا.
   - حفظ البريد الذي كتبه المستخدم محليًا لتسهيل المحاولة التالية.
   - إضافة زر "نسيت كلمة السر؟".
   - في Supabase يتم إرسال رابط إعادة تعيين كلمة المرور إلى البريد.

## ملف SQL المطلوب تشغيله

شغّل الملف التالي من Supabase SQL Editor:

`supabase/sql/patches/007_login_punch_gps_layout_fix.sql`

## ملاحظة مهمة

بعد استبدال النسخة افتح:

`tools/reset-cache.html`

ثم أعد تشغيل النظام من Live Server.
